# 🚀 Rocket Boost

A simple 3D physics-based rocket game built with **Unity** and **C#**. Guide the rocket through challenging levels while avoiding obstacles and safely reaching the landing pad.

---

## 🎮 Gameplay

Control a rocket using realistic physics and navigate through increasingly difficult levels.

### Controls

* **Space** – Thrust
* **A** – Rotate Left
* **D** – Rotate Right
* **Esc** – Quit Game (Build Only)

---

## ✨ Features

* Physics-based rocket movement using Rigidbody
* Smooth camera follow
* Rocket rotation controls
* Thrust and collision sound effects
* Particle effects for engine thrust
* Multiple levels
* Automatic level progression
* Scene reload on crash
* Oscillating obstacles
* Basic lighting and post-processing
* Prefab-based level design
